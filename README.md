# Chef's Table

A react based website on food ordering and cooking.

## Features

### This project contains some key features of ReactJS such as:

-   **Components**
-   **JSX**
-   **Hooks**

### This project also contains some of the ES6 features also:

-   **Arrow function**
-   **Map**
-   **Destructuring**
-   **Spread operator**
-   **Promises**
-   **Modules**
